How to start our first project


You always should start with one really important tag, !DOCTYPE.
!DOCTYPE is used to tell the computer that, you're working with the newest version of html. If you're working with HTML5,
it's Important to use !DOCTYPE. After you make your !DOCTYPE, always create <.html> tag. The <.html> tag is where you build your site. All your code should be inside the html tag.


Example 1:

     <!DOCTYPE html>

Example 2:

     <!DOCTYPE html>
 <html>
 </html>


Also, you can add language to your html and text,
so google can automatically translate your page.

Example 3:

 <html lang="en">
Also can translate from other languages to English.
If you use multiple languages in your text, you can tell what is the language you used in a single text
or for a single word, so the ranslator will know what to translate.
Example: 
<text lang="bg">Здравей, как си?</text>
<text lang="fr">Bonjour comment vas-tu?</text>
<text lang="zh">你还好吗?</text>
<text>Hello, how are you?</text>

#  if your main language (<html lang="en">)
#  is for example, English.
#  You do not have to tell google
#  what language do you use
#  when you write on English.

Creating the Head


The <.head> is one of the most Important parts in the html.
In the Head you can do lots of things like:

  • Adding the title and a favicon of the page:
  • Importing files(for example, your css file)
  • Writing css in the head & much others...


Adding <.head> and title:

     <!DOCTYPE html>
 <html lang="en-US">
    <head>
        <title>Your Title</title>
    </head>
 </html>

Adding Favicon:

Create folder Images where you'll sort your images:

Create your favicon picture and save it in the folder:

Use this tag:

<link rel="icon" type="image/x-icon" href="nameOfFolder/NameOfPicture.ico">

Paste it in the head

Result:

<!DOCTYPE html>
 <html lang="en-US">
    <head>
        <title>Your Title</title>
        <link rel="icon" type="image/x-icon" href="nameOfFolder/NameOfPicture.ico">
    </head>
 </html>

HTML <.meta> charset Attribute

Definition and Usage

The charset attribute specifies the character encoding for the HTML document.

The HTML5 specification encourages web developers to use the UTF-8 character set, which covers almost all of the characters and symbols in the world!


Example 1:

<head>    <meta charset="UTF-8"></head>


Example head:

<!DOCTYPE html><html lang="en-US">    <head>        <meta charset="UTF-8">        <title>            Your Title        </title>        <link rel="icon" type="image/x-icon" href="nameOfFolder/NameOfPicture.ico">    </head></html>
